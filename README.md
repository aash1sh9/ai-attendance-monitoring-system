# AI Attendance Monitoring System

An AI-powered workflow built using n8n and OpenAI API to monitor student attendance and identify at-risk students automatically.

## Features
- Daily attendance checks
- At-risk student detection
- Database filtering
- AI workflow automation

## Tech Stack
- n8n
- OpenAI API
- Automation Workflows
